# Videogames interactive Atlas

En esta web app podrás buscar entre mas de 50.000 videojuegos, navegar entre paginas de la app , y ordenarlos o filtrarlos según sus características, también podrás crear tu propio registro de un nuevo videojuego.

In this web app, you'll be able to search among more than 50,000 video games, navigate through the app's pages, and sort or filter them according to their features. You'll also be able to create your own record for a new video game.

## Tecnologías Implementadas - Tech Stack

**Language:** Javascript

**Client:** React, Redux, ModuleCSS

**Server:** Node, Nodemon, Express, sequelize

**Database:** Postgresql

## Características - Features

-   Barra de búsqueda
-   Filtros y ordenamiento
-   Conexion con API y Base de datos
-   Responsividad
